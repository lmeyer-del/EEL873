Descrição Informal

O app consiste de um diário escolar digital, semelhante aos físicos. Cada professor deve ter uma conta onde consiga adicionar os alunos na turma. O app deve ser capaz de calcular a frequência de cada aluno e reprovar automaticamente o aluno com maior porcentagem de falta que o estabelecido pelo professor. Além disso, deve calcular uma média ponderada entre uma quantidade X de avaliações impostas pelo professor, onde o professor indica o peso de cada avaliação, aprovando automaticamente o aluno com média maior ou igual a um valor M estabelecido pelo professor.. Também deve impedir que o aluno assuma uma média maior que 10.

Valor Entregue

O sistema via DIMINUIR O ESFORÇO e ACELERAR O TEMPO. Diminui o esforço porque o professor não precisará mais calcular a média dos alunos, nem contabilizar faltas.
